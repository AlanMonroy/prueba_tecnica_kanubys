Buenos dias, el aplicativo tiene muchos puntos de mejora pero espero sea lo adecuado para mostrar mis habilidad basicas usando React

Para la conexion a base de datos puede usar la base y el user por defecto de Postgres y usar su password. Esta configuracion esta en el json configDbConection.

Hacer la instalacion de dependencias tanto en backend como en frontend (npm install o pnpm install)
Encender el servidor usando el script "dev" en backend. Esto creara las correspondientes tablas en su base con algunas tareas.
Encender el cliente usando el script "dev" en frontend. Para acceder requerira crear un usuario.

La representacion de base de datos se encuentas en los json: taskRepresentacion.json y userRepresentacion.json de acuerdo los modelos correspondientes,
los modelos se encuentras en el backend, en la carpeta 'models'
